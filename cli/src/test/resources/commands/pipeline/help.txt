Usage: agogosctl pipelines [-hvV] [-o=<output>] [COMMAND]
Interact with pipelines
  -h, --help              Show this help message and exit.
  -o, --output=<output>   Output format, valid values: plain, yaml, json,
                            default: plain.
  -v, --verbose
  -V, --version           Print version information and exit.
Commands:
  describe, desc, d  describe pipeline
  list, l            list pipelines
  run                run a pipeline
